The default avatar group component:

<Playground
  scope={{ AvatarGroup, Avatar }}
  code={`() => {
  const data = [
    {src: "/avatar.webp"},

# Max count

The maximum number of avatars allowed in the list can be customized.

<Playground
  scope={{ AvatarGroup, Avatar }}
  code={`() => {
  const data = [
    {src: "/avatar.webp"},
    {src: "/avatar2.webp"},
    {src: "/avatar3.webp"},
    {src: "/avatar4.webp"},
    {src: "/avatar5.webp"},
    {src: "/avatar6.webp"},
  ];
  return <AvatarGroup data={data} maxCount={3} size={48} />
};`}
/>
