import Avatar, {AvatarSize} from "../../src/components/avatar";
import Playground from "../../docs/playground";

# Default

The avatar component displays the image by default:

<Playground
  scope={{ Avatar }}
  code={`<Avatar src="/avatar.webp" alt="Jason Smith" />`}
/>

# Fallback

If the image is not specified or cannot be loaded, the avatar component can fallback and display a textual representation.
<Playground
  scope={{ Avatar }}
  code={`<Avatar alt="Jason Smith" />`}
/>

# Appearance

Use the appearance attribute to change the avatar shape.

<Playground
  scope={{ Avatar }}
  code={`<Avatar alt="Jason Smith" appearance="square" />`}
/>

# Size

Use the size attribute and the size enumeration to change the avatar size.

<Playground
  scope={{ Avatar, AvatarSize }}
  code={`() => (
    <>
        <Avatar src="/avatar.webp" size={AvatarSize.tiny} />
        <Avatar src="/avatar.webp" size={AvatarSize.small} />
        <Avatar src="/avatar.webp" size={AvatarSize.medium} />
        <Avatar src="/avatar.webp" size={AvatarSize.large} />
        <Avatar src="/avatar.webp" size={AvatarSize.xlarge} />
        <Avatar src="/avatar.webp" size={AvatarSize.xxlarge} /> 
    </>
);`}
/>

Use the size attribute to set a custom avatar size.

<Playground
  scope={{ Avatar }}
  code={`<Avatar src="/avatar.webp" alt="Jason Smith" size={54} />`}
/>
